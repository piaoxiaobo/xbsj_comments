## HTML元素

### 示例代码
```
<!DOCTYPE html>
<html>
  <head>
	<meta />
    <title>元素</title>
  </head>
  <body>
    <h1>元素</h1>
  </body>
</html>
```
HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。
#### HTML元素语法
以开始标签起始，以结束标签终止。
```
<p>：开始标签
</p>：结束标签
```
但并不是所有元素都具有一个结束标签：
```
<meta/>：自结束标签，即从开始标签中结束
```
元素的内容是开始标签与结束标签之间的内容。
